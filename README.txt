#Class Generator

Tool to assist in the creation of classes, enumerations, and interfaces for Java software development
*Requires Java

To use:
1) Run the inputGenerator.jar executable
	- Enter your classes, enums, and interfaces with their respective information
	- Click "Generate Example" if you want to skip this step
	- Click Done when you have entered all the information
	- A file called input.txt will be created. Open this and fill in the remaining information

2) Run the classGenerator.jar executable
	- The .java files corresponding to the information entered previously will be created
